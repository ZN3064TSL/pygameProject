Для запуска игры требуется запустить файл main.py и чтобы папка data была в одной директории с этим файлом.

Цель игры заключается в уничтожении противников, после уничтожения всех противников запускается следующий уровень,
после прохождения всех уровней игра завершается.

В исходном коде представленны следующие сущности:
class Border - класс реализующий работу границ уровня, при создании экземпляра класса принимает на вход
координаты 4 углов границы.

class Platform - класс реализующий работу платформ

class Player - класс реализующий работу персонажа, имеет такие методы как
update - обновление положения спрайта в пространстве,
jump - метод реализующий механику прыжка,
move_left - метод реализующий движение влево,
move_right - метод реализующий движение вправо,
stop_moving_left - метод реализующий остановку персонажа при движении влево,
stop_moving_right - метод реализующий остановку персонажа при движении вправо

class Bullet - класс реализующий работу пуль, принимающий на вход координаты, направление и группу противников, имеет такие методы как
update - обновление положения спрайта в пространстве,

class Enemy - класс реализующий работу противника, принимающий на вход координаты, имеет такие методы как
update - обновление положения спрайта в пространстве

функция crop_image - необходима для обрезки изображения фона, принимает на вход координаты левого верхнего угла,
ширину и высоту.

функция draw_restart_button - необходима для отрисовки надписи Press Enter to Restart при смерти персонажа
